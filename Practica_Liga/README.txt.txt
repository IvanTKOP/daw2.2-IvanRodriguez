-- PROYECTO LIGA --

Este proyecto consiste en crear una aplicación que permita guardar información
de equipos que compitan en la liga española de fútbol.

Como por ejemplo: 
- Nombre del club
- Primer entrenador
- Red social oficial del equipo
- Categoría en la que compiten 

El proyecto consta de:
- 8 archivos PHP separados en EQUIPO y CATEGORÍA:
	*LISTADO (ver listado de categorías/equipos creados)
	*FICHA (permite crear o ver la ficha del equipo/categoría según la id mandada)
	*GUARDAR (guarda las creaciones o modificaciones realizadas)
	*ELIMINAR (permite eliminar un equipo o categoría)
- 1 archivo PHP especial para "Favoritos"
- 1 archivo PHP de sentencias para la base de datos
- 1 archivo SQL que contiene la base de datos
